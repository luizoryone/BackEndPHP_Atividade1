Atividade 1: Navegadores, História da Internet e Protocolo TCP/IP

Introdução

Esta atividade visa apresentar conceitos fundamentais relacionados ao funcionamento da internet, destacando a evolução dos navegadores web, a história da internet e o papel do protocolo TCP/IP. O material está organizado para auxiliar no entendimento básico e serve como base para pesquisas futuras.

1. Navegadores

Os navegadores são softwares essenciais que permitem o acesso e a navegação pelas páginas e serviços da internet. Eles interpretam o código HTML, CSS e JavaScript para apresentar conteúdos ricos e interativos ao usuário.

Resenha dos termos:

  • Browser: Programa que interpreta e exibe páginas web.
  • Rendering engine: Motor que processa o código e renderiza as páginas.
  • Cookies: Pequenos arquivos que armazenam dados do usuário para personalização.
  • Extensões: Plugins que adicionam funcionalidades extras ao navegador.

2. História da Internet

A internet surgiu como um projeto militar nos anos 1960, com o objetivo de criar uma rede resistente a falhas, que conectasse diferentes centros de pesquisa e universidades. O desenvolvimento evoluiu até a popularização mundial que conhecemos hoje.

Resenha dos termos:

  • ARPANET: Rede experimental dos EUA, precursora da internet.
  • Protocolo TCP/IP: Conjunto de regras para comunicação entre computadores.
  • World Wide Web: Sistema de hipertexto criado por Tim Berners-Lee em 1989.
  • ISP: Provedor de serviços de internet que conecta usuários à rede global.

3. Protocolo TCP/IP

O protocolo TCP/IP é a base da comunicação na internet. Ele define como os dados são divididos, endereçados, transmitidos e recebidos, garantindo que a informação chegue corretamente ao destino.

Resenha dos termos:

  • TCP (Transmission Control Protocol): Garante a entrega correta e ordenada dos pacotes de dados.
  • IP (Internet Protocol): Responsável pelo endereçamento e roteamento dos pacotes.
  • Packet: Unidade básica de dados transmitida na rede.
  • Handshake: Processo de estabelecimento da conexão entre dois dispositivos.

Glossário

Termo Definição
Browser Software que permite a visualização e interação com páginas web.
ARPANET Rede de computadores experimental dos EUA, antecessora da internet.
TCP/IP Conjunto de protocolos que controla a transmissão de dados na internet.
Hipertexto Texto com links que conectam a outras páginas ou documentos.
ISP Empresa que fornece acesso à internet aos usuários.
